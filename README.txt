MYMAP
A phone-first Seattle walking and public-transit pin map for a UW student.

FILES TO UPLOAD TO GITHUB PAGES

FIRST USE
1. Open the published HTTPS site on her phone.
2. Press + and tap the apartment location.
3. Choose the Home category and save it.
4. Add classrooms, grocery stores, cafes, pharmacies, study spots, and favorite places.
5. Open a saved pin and press Walk or Transit for live directions in Google Maps.
6. Use the HS menu to make regular JSON backups.

IMPORTANT
- Pins and notes are stored in that browser on that device, not in an online account.
- Clearing site/browser data removes pins unless a backup was downloaded.
- Search and transit-stop overlays require internet access.
- Map and transit-stop data may be incomplete.
- Host through HTTPS (such as GitHub Pages) for location and install features.
